# 💰 Smart Expense Tracker

A fully functional Python-based **Smart Expense Tracker** application with GUI, database storage, analytics, and CSV export features.

This project helps users manage their daily expenses efficiently with visual reports and categorized tracking.

---

# 🚀 Features

✅ User Registration & Login  
✅ Expense Management  
✅ SQLite Database Integration  
✅ Expense Categorization  
✅ Analytics & Pie Charts  
✅ CSV Report Export  
✅ Interactive GUI using Tkinter  
✅ Modular Project Structure  
✅ Beginner-Friendly Code  

---

# 🛠️ Technologies Used

- Python
- Tkinter
- SQLite3
- Pandas
- Matplotlib
- CSV
